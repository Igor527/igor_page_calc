Файлы для статического сайта (GitHub Pages). Все данные админки копятся здесь и при включённой синхронизации пушатся в репо (быстрый деплой).

Автосинхронизация (рекомендуется):
  1. Войдите в режим админа: https://urbanplanner.page/welcome_me (вход по Google/GitHub/email).
  2. На главной внизу откройте «Синхронизация с GitHub»: укажите owner, repo, ветку и Personal Access Token (права repo).
  3. Сохраните и нажмите «Проверить подключение». Дальше все изменения ниже автоматически пушатся в репо (деплой через GitHub Actions подхватит новые коммиты).

Файлы (создаются автосинком или вручную):
  calculators.json — опубликованные калькуляторы и отчёты (редактор → Опубликовать)
  posts.json       — посты блога (в т.ч. с картинками обложек)
  notes.json       — заметки и папки
  dictionary.json  — словарь (переводы)
  layouts.json     — порядок кнопок/секций на главной
  planner.json     — задачи планировщика (Гантт/календарь)

Доступ к разделам (админка скрыта с публичного сайта): редактор, заметки, словарь, планировщик — только после входа по /welcome_me. Публично видны: главная, калькуляторы, блог.

Без этих файлов данные берутся только из localStorage.
