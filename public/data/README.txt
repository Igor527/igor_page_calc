Файлы для статического сайта (GitHub Pages). Можно заполнять вручную или включить автосинхронизацию.

Автосинхронизация (рекомендуется):
  1. Войдите в режим админа (главная → «Войти в режим админа»).
  2. Внизу откройте «Синхронизация с GitHub»: укажите owner, repo, ветку и Personal Access Token (права repo).
  3. Сохраните и нажмите «Проверить подключение». После этого изменения в заметках, блоге, словаре, калькуляторах и порядке окон на главной будут автоматически пушиться в репо.

Файлы (создаются автосинком или вручную):
  calculators.json — опубликованные калькуляторы (редактор → Опубликовать)
  posts.json — посты блога
  notes.json — заметки
  dictionary.json — словарь (переводы)
  layouts.json — порядок кнопок/секций на главной и других страницах

Без этих файлов данные берутся только из localStorage.
