# Инструкции проекта: Igor.Page Engine (v1.0)

## 1. Концепция
Универсальный визуальный конструктор инженерных расчетов. 
Интерфейс разделен на:
- **Left Panel (Blueprint)**: Техническая настройка логики (блоки ввода, таблицы, скрытые формулы).
- **Right Panel (Report Editor)**: Визуальный редактор отчета (текст как в Word + вставки формул из Blueprint).

## 2. Бизнес-логика (Ядро)
- **State Management**: Единый объект `store`, где ключи — это `id` блоков, а значения — их текущие данные.
- **Reactive Engine**: При изменении любого `INPUT` или `TABLE_LOOKUP`, все зависимые `FORMULA` пересчитываются через `math.js`.
- **Chain Calculation**: Блок 5 может использовать результат Блока 2.

## 3. Реестр Блоков (Schema)
1. `INPUT`: { id, label, type: 'number'|'select', unit, value }
2. `CONSTANT`: { id, value }
3. `TABLE_LOOKUP`: { id, data: any[], key_col, target_col, selected_key }
4. `FORMULA`: { id, expression, template: string } // Пример template: "S {area} * K {k} = {result}"
5. `TEXT`: { content: string, style: 'h1'|'p' }
6. `CONDITION`: { id, if_exp, then_id, else_id }

## 4. Требования к коду
- Next.js 14+ (App Router), TypeScript, Tailwind CSS.
- Библиотеки: `mathjs`, `lucide-react` (иконки), `zustand` (стейт).
- Приватность: Репозиторий приватный, деплой на Vercel/Cloudflare.
