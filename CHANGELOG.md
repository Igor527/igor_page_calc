# 🎉 Обновление 2.1 — Полноценные Dynamo-ноды с портами и связями

**Дата:** 16 февраля 2026

---

## 🌟 Что нового?

### 1. 🎨 Полноценный Dynamo-style интерфейс

Теперь это **настоящие ноды** как в Autodesk Dynamo!

**Ключевые возможности:**
- ✅ **Порты ввода/вывода** — синие кружки слева (входы), зелёные справа (выходы)
- ✅ **Визуальные связи** — SVG линии между нодами показывают поток данных
- ✅ **Drag & Drop соединений** — тяните линию от выхода к входу для создания связи
- ✅ **Inline-редактирование** — двойной клик для изменения названия, клик для редактирования значений
- ✅ **Автоматические dependencies** — формулы обновляются при создании связей

### 2. 🆕 Новый блок: Table Viewer (Просмотр таблицы)

Промежуточный блок для работы с табличными данными:
- Выбор таблицы из выпадающего списка
- Выбор столбца для просмотра
- Порты: вход `dataSource`, выходы `column`, `row`, `cell`
- Можно подключить к формулам для вычислений

**Пример использования:**
```
data_table → table_viewer (column: "price") → formula (price * 1.2)
```

### 3. 🔗 Визуальные связи между нодами

**Как создать связь:**
1. Наведите на **зелёный порт** (выход) первой ноды
2. Зажмите **левую кнопку мыши**
3. Тяните линию к **синему порту** (вход) второй ноды
4. Отпустите кнопку
5. Связь создана! ✨

**Как удалить связь:**
- Кликните на линии связи
- Подтвердите удаление

### 4. ✏️ Inline-редактирование без всплывающих окон

Всплывающее окно PropertyEditor **убрано** — теперь всё редактируется прямо на нодах:
- **Двойной клик** на название → редактировать label
- **Клик** на числовое поле → изменить значение
- **Изменения применяются мгновенно**

## 📖 Как пользоваться?

### Создание ноды
```
ПКМ на канвасе → Выбрать тип → Нода создастся
```

### Создание связи
```
ЛКМ на выходе (зелёный) → Тянем к входу (синий) → Отпускаем
```

### Редактирование
```
Двойной клик на название → Редактировать
Клик на поле → Изменить значение
```

### Удаление связи
```
Клик на линии → Подтвердить
```

---

## 💡 Примеры использования

### Пример 1: Простой калькулятор
```
1. ПКМ → Ввод (a = 10)
2. ПКМ → Ввод (b = 20)
3. ПКМ → Формула (formula: a + b)
4. Соединить: a.output → formula.a
5. Соединить: b.output → formula.b
→ Результат: 30
```

### Пример 2: Работа с таблицей
```
1. ПКМ → Таблица (data_table)
2. ПКМ → Просмотр таблицы (table_viewer)
3. Соединить: table.output → viewer.dataSource
4. В viewer выбрать столбец
5. ПКМ → Формула
6. Соединить: viewer.column → formula.input
→ Формула получает данные столбца
```
