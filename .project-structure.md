# Структура папок

- `/src/app` — Роутинг (/, /admin, /calc/[id])
- `/src/components/editor`
  - `BlueprintPanel.tsx` — Левая панель (логика)
  - `ReportPanel.tsx` — Правая панель (визуальный отчет)
  - `PropertyEditor.tsx` — Настройки выделенного блока
- `/src/lib`
  - `engine.ts` — Расчетный движок на math.js
  - `store.ts` — Состояние калькулятора (Zustand)
- `/src/types`
  - `blocks.ts` — Интерфейсы всех типов блоков
- `/src/data` — Примеры JSON-схем для тестов
