# Правила оформления UI

## Отступ контента от хедера

- **Везде учитывать отступ от хедера:** использовать переменную **`var(--content-offset-from-header)`**.
- Отступ задан **в 2 раза больше**, чем высота хедера: `--content-offset-from-header: calc(var(--site-header-height) * 2)`.
- Высота хедера: `--site-header-height` (по умолчанию 56px). Итого отступ контента от верха страницы — 112px.

### Где применять

- При расчёте высоты контентной области под хедером и футером использовать:
  - `height: calc(100vh - var(--content-offset-from-header) - var(--site-footer-height));`
- Не задавать фиксированные отступы «от хедера» вручную — использовать переменную, чтобы при изменении правила обновлялось везде.
