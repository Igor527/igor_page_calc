# Dynamo-Style Node Editor — Руководство пользователя

## 🎨 Что нового в версии 2.1?

Полноценный Dynamo-подобный интерфейс с **нодами, портами и визуальными связями**!

---

## 🌟 Основные возможности

### 1. Ноды с портами

Каждая нода имеет:
- **Входные порты** (слева, синие кружки) — для приёма данных
- **Выходные порты** (справа, зелёные кружки) — для передачи данных

### 2. Визуальные связи

Соединяйте ноды, создавая поток данных:
- Линии между нодами показывают зависимости
- Данные текут от выхода одной ноды к входу другой

### 3. Inline-редактирование

Редактируйте параметры прямо на нодах:
- **Двойной клик** на название ноды — редактирование label
- **Клик** на числовое поле — изменить значение
- **Изменения применяются мгновенно**

---

## 📖 Как пользоваться?

### Создание ноды

1. **Правая кнопка мыши (ПКМ)** на пустом месте канваса
2. Откроется меню с типами нод
3. Выберите нужный тип
4. Нода создастся в месте клика

**Доступные типы нод:**
- 🔵 **Ввод** (input) — числовые/текстовые поля
- 🟠 **Формула** (formula) — математические выражения
- 🟢 **Текст** (text) — текстовые блоки
- 🟣 **Константа** (constant) — неизменяемые значения
- 🔴 **Таблица** (data_table) — табличные данные
- 🟢 **Просмотр таблицы** (table_viewer) — выбор столбцов из таблицы ✨ НОВИНКА!
- 🔵 **Выбор из таблицы** (select_from_table)
- 🔵 **Выбор из объекта** (select_from_object)
- 🔴 **Условие** (condition) — if/then/else логика
- 🟡 **График** (chart) — визуализация данных
- 🟢 **Вывод** (output) — форматированный вывод
- 🔴 **Кнопка** (button) — действия
- 💗 **Изображение** (image)
- 🔵 **Группа** (group) — контейнер

### Перемещение ноды

- **Левая кнопка мыши + перетаскивание** — переместить ноду
- Ноды можно располагать в любом месте

### Редактирование ноды

**Двойной клик:**
- **На название ноды** — редактировать label/название

**Inline-поля:**
- **Input** — изменить значение по умолчанию
- **Formula** — ввести формулу (например: `a + b`)
- **Constant** — изменить значение константы
- **Table Viewer** — выбрать таблицу и столбец

**Результат вычисления** отображается в зелёной области внизу ноды:
```
= 42
```

### Создание связи между нодами

1. **Наведите мышь** на **выходной порт** (зелёный кружок справа) ноды-источника
2. **Зажмите левую кнопку мыши** на порту
3. **Тяните линию** к **входному порту** (синий кружок слева) ноды-приёмника
4. **Отпустите кнопку мыши** над входным портом
5. Связь создана! Линия соединяет ноды

**Удаление связи:**
- **Клик** на линии связи
- Подтвердите удаление

**Типы портов:**
- **output** → **input** — передача значения
- Формула автоматически получает зависимость

### Пример: Простой калькулятор

**Шаг 1: Создайте входы**
```
ПКМ → Ввод (a)
ПКМ → Ввод (b)
```

**Шаг 2: Создайте формулу**
```
ПКМ → Формула
Двойной клик на "Формула" → Введите: a + b
```

**Шаг 3: Соедините ноды**
```
Выход a (зелёный порт) → Вход формулы "a"
Выход b (зелёный порт) → Вход формулы "b"
```

**Шаг 4: Измените значения**
```
Клик на поле "a" → Введите 10
Клик на поле "b" → Введите 20
→ Формула покажет: = 30
```

### Пример: Работа с таблицами

**Шаг 1: Создайте таблицу**
```
ПКМ → Таблица (data_table)
```

**Шаг 2: Создайте просмотр таблицы (✨ новинка!)**
```
ПКМ → Просмотр таблицы (table_viewer)
```

**Шаг 3: Соедините**
```
Выход таблицы → Вход "dataSource" просмотра
```

**Шаг 4: Выберите столбец**
```
В просмотре таблицы:
1. Выберите таблицу из выпадающего списка
2. Выберите столбец
→ Просмотр покажет данные столбца
```

## 🎯 Навигация

### Панорамирование канваса

- **Shift + Левая кнопка мыши** — перемещение всей области
- **Средняя кнопка мыши** — перемещение всей области

### Удаление ноды

- Кнопка **×** в правом верхнем углу ноды
- Подтверждение удаления

### Выбор ноды

- **Клик левой кнопкой** на ноду
- Выбранная нода подсвечивается и увеличивается

### Отмена выбора

- **Клик левой кнопкой** на пустом месте

## 💡 Советы и трюки

### ✅ Рекомендуется

- **Используйте понятные имена нод** (двойной клик для редактирования)
- **Группируйте связанные ноды** визуально рядом
- **Соединяйте ноды слева направо** (как в Dynamo)
- **Проверяйте результаты** в зелёной области внизу ноды
- **Используйте Table Viewer** для работы с табличными данными

### ❌ Избегайте

- Создания циклических зависимостей (нода → нода → первая нода)
- Слишком длинных формул (лучше разбить на части)
- Пустых формул без dependencies

## 🆕 Новый блок: Table Viewer (Просмотр таблицы)

**Зачем нужен?**
Промежуточный блок для работы с таблицами — позволяет выбирать столбцы и строки из таблиц.

**Как использовать?**

1. Создайте ноду **data_table** с данными
2. Создайте ноду **table_viewer**
3. Соедините выход таблицы со входом table_viewer
4. В table_viewer выберите:
   - Источник (таблица)
   - Столбец для просмотра
5. Выход table_viewer можно подключить к формуле

**Порты:**
- **Вход:** `dataSource` (таблица)
- **Выходы:** `column`, `row`, `cell` (данные)

**Пример использования:**
```
data_table (Prices) → table_viewer → formula (price * qty)
```

## 🐛 Решение проблем

### Нода не показывает результат

- Проверьте, что все входные порты подключены
- Проверьте формулу на ошибки
- Убедитесь, что dependencies корректны

### Связь не создаётся

- Убедитесь, что тянете от **выходного** (зелёного) порта к **входному** (синему)
- Проверьте, что порты поддерживают соединение
- Наведите мышь точно на порт

### Ноды накладываются

- Переместите ноды, используя drag & drop
- Используйте панорамирование (Shift + ЛКМ) для навигации

### Линия связи не удаляется

- Кликните **точно на линии** (не на портах)
- Или удалите ноду с одной стороны — связь удалится автоматически
