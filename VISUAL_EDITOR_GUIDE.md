# Руководство по визуальному редактору (стиль Dynamo)

## Обзор

Новый визуальный редактор создан по образцу Autodesk Dynamo — визуального программирования с блоками-узлами.

## Основные возможности

### 1. Двухпанельный режим
- **Левая панель** — рабочая область с визуальными блоками
- **Правая панель** — отчёт с результатами расчётов в реальном времени

### 2. Создание блоков

#### Контекстное меню (ПКМ)
1. Кликните **правой кнопкой мыши** на любое место рабочей области
2. Откроется контекстное меню со всеми типами блоков
3. Выберите нужный тип блока
4. Блок создастся в месте клика

**Доступные типы блоков:**
- 🔵 **Ввод** (input) — поля ввода данных
- 🟠 **Формула** (formula) — математические выражения
- 🟢 **Текст** (text) — текстовые элементы
- 🟣 **Константа** (constant) — неизменяемые значения
- 🔵 **Поиск в таблице** (table_lookup)
- 🔴 **Таблица** (data_table)
- 🟡 **График** (chart)
- 🔵 **Выбор из таблицы** (select_from_table)
- 🔵 **Выбор из объекта** (select_from_object)
- 🔴 **Условие** (condition)
- 🔵 **Группа** (group)
- 🟢 **Вывод** (output)
- 💗 **Изображение** (image)
- 🔴 **Кнопка** (button)

### 3. Работа с блоками

#### Перемещение блоков
- **Левая кнопка мыши + перетаскивание** — переместить блок
- Блоки можно располагать в любом месте канваса

#### Выбор блока
- **Клик левой кнопкой** на блок — выбрать блок
- Выбранный блок подсвечивается рамкой и увеличивается масштаб
- Автоматически открывается **всплывающее окно свойств** справа

#### Редактирование свойств
- После выбора блока справа появляется **плавающая панель свойств**
- В панели можно изменить:
  - Заголовок (label)
  - Специфичные параметры (формула, значения, источники данных и т.д.)
- Все изменения применяются мгновенно
- Закрыть панель свойств — кнопка **×** в правом верхнем углу панели

#### Удаление блока
- Кнопка **×** в правом верхнем углу карточки блока
- Подтверждение удаления через диалоговое окно

### 4. Навигация по канвасу

#### Панорамирование (перемещение области просмотра)
- **Shift + Левая кнопка мыши** — перемещение всей рабочей области
- **Средняя кнопка мыши** — перемещение всей рабочей области

#### Визуальная сетка
- Фон рабочей области содержит сетку 20×20 пикселей
- Помогает ориентироваться и выравнивать блоки

### 5. Визуальное представление блоков

#### Структура блока-карточки
```
┌─────────────────────────────┐
│  [Тип блока]            [×] │ ← Заголовок с цветом типа
├─────────────────────────────┤
│  Название блока             │ ← Label
│  ID: block_xyz123           │ ← Идентификатор
│                             │
│  [Дополнительная информация]│ ← Формулы, значения
│                             │
│  Результат: 42              │ ← Вычисленное значение
└─────────────────────────────┘
```

#### Цветовая кодировка
Каждый тип блока имеет свой цвет рамки:
- 🔵 Синий — блоки ввода и выбора
- 🟠 Оранжевый — формулы и вычисления
- 🟢 Зелёный — текст и вывод
- 🟣 Фиолетовый — константы
- 🔴 Красный — условия, кнопки, таблицы
- 🟡 Жёлтый — графики

### 6. Панель результатов (правая панель)

#### Отчёт
- Интерактивный отчёт с результатами всех вычислений
- Обновляется автоматически при изменении значений

#### Таблица результатов
- **Блок** — название блока
- **Значение** — результат вычисления
- Отображается в реальном времени

#### Отладка
- Секция с JSON выбранного блока для отладки
- Помогает понять структуру блока

### 7. Горячие клавиши

| Действие | Клавиша/Мышь |
|----------|-------------|
| Создать блок | ПКМ на канвасе |
| Выбрать блок | ЛКМ на блоке |
| Переместить блок | ЛКМ + перетаскивание |
| Панорамирование | Shift + ЛКМ или СКМ + перетаскивание |
| Удалить блок | × на карточке блока |
| Закрыть свойства | × на панели свойств |

### 8. Работа с формулами

#### Создание формулы
1. Создайте блок типа **Формула** через ПКМ
2. Выберите блок
3. В панели свойств введите math.js выражение в поле "Формула"
4. Укажите зависимости (ID других блоков через запятую)

#### Автодополнение в формулах
- Нажмите `{` в поле формулы — откроется список доступных блоков
- Кликните на блок в списке — его ID вставится в формулу

#### Пример
```
Блок "a" (input): defaultValue = 10
Блок "b" (input): defaultValue = 20
Блок "sum" (formula): formula = "a + b", dependencies = ["a", "b"]
→ Результат sum: 30
```

### 9. Советы и рекомендации

✅ **Рекомендуется:**
- Используйте понятные названия блоков (label)
- Группируйте связанные блоки визуально рядом
- Регулярно проверяйте панель результатов
- Используйте разные типы блоков для разных задач

❌ **Не рекомендуется:**
- Создавать слишком много блоков без структуры
- Использовать дублирующиеся ID (система предотвращает это автоматически)
- Оставлять пустые формулы или некорректные зависимости

### 10. Сохранение и экспорт

Для сохранения калькулятора используйте кнопки в верхней панели навигации:
- **Редактор** — текущая страница
- **Ревью** — отправить на проверку администратору

## Решение проблем

### Блок не показывает результат
- Проверьте формулу на ошибки
- Убедитесь, что зависимости указаны корректно
- Проверьте, что все зависимые блоки существуют

### Всплывающее окно свойств не открывается
- Убедитесь, что блок выбран (подсвечен)
- Кликните на блок ещё раз левой кнопкой мыши

### Блоки накладываются друг на друга
- Переместите блоки, используя drag & drop
- Используйте панорамирование (Shift + ЛКМ) для навигации
